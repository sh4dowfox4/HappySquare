## 📖 О игре

**HappySquare** — это увлекательная инди-игра, которая подарит тебе море позитива! Помоги нашему синему кубику пройти через все препятствия и собрать максимальное количество очков. Простая механика, но затягивающий геймплей!

## ✨ Особенности

- 🎯 Простое и интуитивное управление
- 🎨 Приятная графика с плавной анимацией
- 🎵 Оптимизированная производительность
- 🌟 Кроссплатформенность (Linux & Windows)
- 💡 Лёгкий и понятный исходный код

## 🛠️ Технологии

Проект разработан с использованием:
- **C++17** — современный стандарт C++
- **Raylib** — простая и мощная игровая библиотека

## 📦 Установка и сборка

### 🐧 Linux
```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/HappySquare.git
cd HappySquare

# Запустите скрипт сборки
chmod +x build.sh
./build.sh

# Запустите игру
./HappySquare
