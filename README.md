# 📦Ручная установка и сборка 

## 🐧 Linux
#### Клонируйте репозиторий
```bash
git clone https://github.com/sh4dowfox4/HappySquare.git
cd HappySquare
```
#### Запустите скрипт сборки
```bash
chmod +x build.sh
./build.sh
```
#### Запустите игру
```bash
./HappySquare
```

## ♿ Windows
##### ! Нужен компилятор MinGW !
#### Клонируйте репозиторий
```batch
git clone https://github.com/sh4dowfox4/HappySquare.git
cd HappySquare
```
#### Запустите скрипт сборки
```batch
build.bat
```
#### Запустите игру
```batch
HappySquare.exe
```
